docker 使用

发布： 2011-02-27 分类： docker

制作镜像

两种方式

  1. docker commit命令
  2. 使用docker build和Dockerfile文件

镜像打包

1.镜像打包

docker save -o /root/xxx.tar  <name>
2.导入镜像

docker load -i /root/xxx.tar

容器打包

1.容器打包

docker export -o /root/xx.tar  <name>
2.导入容器

docker import xx.tar <name>:latest
